The Transmission Control Protocol/Internet Protocol, TCP/IP, Model is a practical framework as opposed to OSI's theoretical model. It was developed by the Defense Advanced Research Projects Agency (DARPA) and consists of 4 layers

The TCP/IP Model is divided into 4 distinct layers:

  • Application Layer (Layer 1) – Serves as the user's interface to communicate through the network.
  • Transport Layer (Layer 2) – Handles data flow and error checking.
  • Internet Layer (Layer 3) – Involved in logical addressing and routing of payloads.
  • Network Access Layer (Layer 4) – Facilitates physical movement of data over the network medium.

The TCP/IP has 4 layers which are traversed by data as it is processed. Data typically moves from the Application Layer down to the Network Access layer, where headers are appended for transmission. The receiver removes these headers as the data moves from bottom to top. This process ensures organized and reliable communication.

Each TCP/IP layer uses specific protocol
The Application Layer includes HTTP, FTP, SMTP, and DNS
The Transport Layer uses TCP and UDP to manage data delivery.
The Internet Layer uses IP and ICMP for addressing and routing.
The Network Access Layer includes Ethernet and ARP for physical transmission and hardware addressing.
