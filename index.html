There are two common network models to represent the flow of data Communication: The OSI Model and the TCP/IP MODEL

Model Comparison

The table below summarises key difference between the two network models.

OSI MODEL TCP/IP MODEL
7 Layers 4 Layers
No specified protocols used Uses specified protocols such as TCP, UDP, IP, etc.
Developed by ISO for standardization, more theoretical Developed by DoD for Internet use, more practical
Has a separate presentation and session layer Does not have a separate presentation and separate layer
